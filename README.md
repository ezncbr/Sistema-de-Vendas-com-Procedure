# Sistema-de-Vendas-com-Procedure
O objetivo foi criar um sistema simples de vendas usando PostgreSQL, onde é possível cadastrar produtos e registrar vendas por meio de uma procedure.
O que foi feito

Criação das tabelas produtos e vendas
Inserção de produtos iniciais
Procedure realizar_venda que valida o estoque, registra a venda e atualiza a quantidade disponível automaticamente
